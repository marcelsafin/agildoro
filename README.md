# FocusLoop

FocusLoop is a persistent, mobile-first Agile planning tool and Focus Timer (PWA).

## Features

- **Kanban Board**: Manage tasks with subtasks and drag-and-drop support.
- **Focus Timer**: Pomodoro-style timer with visual feedback.
- **Offline First**: All data is stored locally in your browser.
- **Mobile Optimized**: Designed to feel like a native app on your phone.

## Run Locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open http://localhost:3000
